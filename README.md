# Attestation Hub

Plateforme de gestion des demandes d'attestations pour les employés de G-Innovation.

## Fonctionnalités

- **Tableau de bord** : Vue d'ensemble des demandes d'attestations
- **Gestion des demandes** : Visualisation, téléchargement et gestion des statuts
- **Génération de PDF** : Création automatique de certificats de travail et d'attestations de travail
- **Authentification** : Connexion sécurisée avec Google OAuth
- **Gestion des utilisateurs** : Ajout et gestion des employés

## Installation

1. Cloner le dépôt
2. Installer les dépendances :
   ```bash
   npm install
   ```
3. Lancer l'application :
   ```bash
   npm run dev
   ```
